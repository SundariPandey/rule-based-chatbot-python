# Rule-Based Chatbot in Python 🤖

This project is a simple **Rule-Based Chatbot built using Python**.
The chatbot responds to user inputs based on predefined rules and conditions.

## Features

* Interactive chatbot conversation
* Typing effect for realistic responses
* Handles basic user questions
* Beginner-friendly Python logic

## Technologies Used

* Python
* Conditional statements
* Functions
* Basic string handling

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Run the Python file:

```
python chatbot.py
```

## Example

User: Hello
Bot: Hi! How can I help you today?

User: What is your name?
Bot: I am a simple rule-based chatbot.

## Purpose

This project was created to practice **Python programming and basic chatbot development**.
